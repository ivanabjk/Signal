# SIGNAL

SIGNAL is a 2D side-scrolling platformer about media literacy and misinformation.

You play as **Kai**, a teenage analyst exploring **The Grid** — a digital city corrupted by **NULL**, a rogue algorithm spreading misinformation. Kai searches for his missing sister **Mia** while being guided by a damaged AI called **ECHO**.

## Gameplay

- 3 unique zones with different visuals and platform layouts
- Enemies in every zone
- Enemies cannot be defeated - only avoided
- 3 lives per zone
- WASD controls only

## Quiz System

Each zone ends with a checkpoint tower locked behind a **five-card sorting quiz**. Players must classify terms into the correct categories to continue.

- Wrong answers give hints
- Unlimited retries
- No life penalty

## Technical

- Uses HTML5 Canvas for gameplay
- Quiz UI uses a simple `div` overlay
- Uses GitHub pages
